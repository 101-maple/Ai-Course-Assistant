# Ai-Course-Assistant

## Overview

A full-stack AI study assistant for uploading course materials, generating summaries, quizzes,  and document-grounded Q&amp;A.

The goal is to build a practical AI learning assistant for college students who need to review lecture notes, PDFs, and other course materials.

## Target Users

College students who wants to study and organize their materials more effectively.

## Core Features

- Uploading course materials as PDFs
- Extract text from PDF
- Generating summaries
- Generating quiz questions
- Ask questions based on uploaded document
- Return answers with source references
